# CPD Guaranésia - Sistema de Chamados

Este é o sistema oficial de Help Desk e Service Desk desenvolvido para o Centro de Processamento de Dados (CPD) da Prefeitura de Guaranésia.

## 🚀 Sobre o Projeto

O sistema foi projetado para otimizar o atendimento de chamados técnicos, permitindo uma gestão eficiente das demandas internas da prefeitura. Com uma interface intuitiva e recursos inteligentes, o sistema facilita a comunicação entre os usuários e a equipe técnica.

## ✨ Principais Funcionalidades

- **Gestão de Chamados:** Abertura, acompanhamento e resolução de tickets de suporte.
- **Inteligência Artificial:** Análise automática de chamados para classificação de prioridade e categoria.
- **Controle de Acesso:** Gerenciamento de usuários, técnicos e administradores com permissões específicas.
- **Painel de Controle:** Dashboards com métricas de desempenho e produtividade da equipe.
- **Histórico e Auditoria:** Log completo de ações para rastreabilidade e segurança.
- **Interface Responsiva:** Otimizado para uso em computadores e dispositivos móveis.

## 🛠 Tecnologias Utilizadas

- **Frontend:** React, TypeScript, Tailwind CSS, Motion (animações), Recharts (gráficos).
- **Backend/Banco de Dados:** Supabase (PostgreSQL + Auth).
- **IA:** Integração com modelos Google Gemini para análise de chamados.

## 📦 Como Rodar o Projeto

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure as variáveis de ambiente necessárias (Supabase, Gemini API, etc.).
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
